# ToDoDo

ToDoDo는 **바탕화면에 고정해서 사용하는 개인 작업 보드**입니다. 
항상 켜 두고 확인하는 데스크톱형 ToDo 앱을 목표로 설계했습니다.

## 주요 기능

- **바탕화면 고정형 보드**
  - 일반 창 뒤에 머무르도록 다시 고정할 수 있습니다.
  - 트레이로 숨기고 다시 복원할 수 있습니다.

- **목록 기반 정리**
  - 작업을 여러 목록으로 나눠 관리할 수 있습니다.
  - 목록은 클릭해서 선택하고, 다시 눌러 바로 이름을 바꿀 수 있습니다.

- **빠른 입력 + 세부 옵션**
  - 새 작업을 추가할 때 제목, 마감일, 우선순위, 반복 일정을 함께 지정할 수 있습니다.
  - 마감일은 DatePicker 캘린더로 선택합니다.

- **스마트 필터**
  - 전체 / 오늘 / 이번 주 / 나중에 필터를 제공합니다.

- **정리 기능**
  - 작업 완료 처리
  - 완료 항목 자동 접기
  - 목록 내부 드래그 앤 드롭 정렬
  - 위/아래 이동 버튼으로 순서 조정

- **설정 패널**
  - 바탕화면 고정 유지
  - 사이드바 표시 여부
  - `Ctrl + Alt + T` 전역 빠른 추가 단축키
  - 완료 항목 자동 접기
  - 백업 내보내기 / 가져오기

- **자동 저장**
  - 종료하거나 재부팅해도 목록과 설정이 유지됩니다.

## 저장 위치

- `%AppData%\ToDoDo\data.json`
- `%AppData%\ToDoDo\settings.json`

## 폰트

실행 파일 기준 상위 경로를 따라 올라가며 `fonts` 폴더를 찾고, 그 아래를 **재귀적으로 탐색**합니다. 
`Pretendard` 계열 폰트를 우선 적용하고, 없으면 시스템 기본 폰트로 대체합니다.

예시:

```text
ToDoDo/
 ├─ fonts/
 │   └─ Pretendard/
 │       ├─ Pretendard-Regular.otf
 │       └─ Pretendard-SemiBold.otf
 └─ bin/
```

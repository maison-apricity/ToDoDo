# ToDoDo

**ToDoDo**는 Windows 데스크톱 환경을 위한 **그룹 기반 작업 관리 프로그램**입니다.  
할 일을 간단히 적어 두는 수준을 넘어, 작업을 분류하고, 진행 상태에 따라 정리하고, 완료된 항목과 기록을 체계적으로 관리할 수 있도록 설계되었습니다.

프로젝트별, 주제별, 업무별로 작업을 나누어 관리할 수 있으며, 현재 진행 중인 일과 완료된 일을 구분해서 볼 수 있습니다. 오래된 완료 항목은 보관함으로 옮겨 정리할 수 있어, 현재 필요한 정보에 집중하면서도 이전 기록은 유지할 수 있습니다. 바탕화면 고정과 트레이 연동도 지원하여, Windows 데스크톱 환경에서 자연스럽게 사용할 수 있는 작업 관리 도구를 지향합니다.

---

## 프로그램의 목적

ToDoDo는 작업을 빠르게 기록할 수 있으면서도, 시간이 지나도 목록을 일관된 구조 안에서 관리할 수 있도록 하는 데 목적이 있습니다.

작업이 많아질수록 단순한 목록만으로는 현재 해야 할 일, 이미 끝난 일, 반복되는 일, 오래된 기록을 구분하기가 어려워집니다. ToDoDo는 이러한 흐름을 정리하기 위해 **그룹**, **진행 상태**, **보관**이라는 구조를 중심으로 작업을 다룹니다. 이를 통해 사용자는 작업을 성격에 맞게 분류하고, 현재 필요한 항목에 집중하면서도 이전 기록을 필요에 따라 다시 확인할 수 있습니다.

---

## 작동 방식

ToDoDo는 작업을 그룹 안에서 관리하고, 각 항목을 진행 상태에 따라 구분하는 방식으로 동작합니다.

각 작업은 하나의 그룹에 속하며, 사용자는 원하는 기준에 따라 그룹을 나누어 사용할 수 있습니다. 작업이 완료되면 완료 영역으로 이동하여 진행 중 목록과 분리되며, 더 이상 현재 목록에 둘 필요가 없는 항목은 보관함으로 이동시켜 장기적으로 정리할 수 있습니다. 이 흐름을 통해 현재 작업, 최근 완료된 작업, 이전 기록이 서로 섞이지 않고 명확하게 구분됩니다.

---

## 주요 특징

### 그룹 중심의 작업 구성
ToDoDo는 하나의 긴 목록 대신, 그룹을 기준으로 작업을 분류합니다.  
이를 통해 프로젝트별, 업무별, 주제별로 작업 맥락을 유지한 채 관리할 수 있습니다.

### 진행 중 · 완료 · 보관의 구분
현재 해야 할 일, 이미 끝난 일, 장기 기록으로 남겨둘 일을 서로 다른 단계로 나누어 관리할 수 있습니다.  
이 구조는 현재 목록의 가독성을 높이면서도 기록을 유지하는 데 도움이 됩니다.

### 빠른 정리와 재구성
작업 추가, 수정, 완료, 복제, 이동, 삭제 같은 흐름을 빠르게 처리할 수 있도록 구성되어 있습니다.  
유사한 작업을 반복적으로 다루는 경우에도 효율적으로 사용할 수 있습니다.

### 데스크톱 중심 사용성
ToDoDo는 모바일형 메모 앱보다, 데스크톱에서 상시 사용되는 작업 패널에 가까운 성격을 갖습니다.  
바탕화면 고정, 트레이 연동, 독립 창 구조를 통해 Windows 환경에 자연스럽게 어울리도록 설계되었습니다.

### 정리된 인터페이스
진행 목록, 완료 목록, 보관함이 분리되어 있어 사용자는 현재 필요한 정보와 기록성 정보를 구분해서 볼 수 있습니다.  
설정창과 보관함도 별도 창으로 제공되어 전체 구조가 명확합니다.

## 주요 기능

ToDoDo는 다음과 같은 작업 관리 기능을 제공합니다.

- 그룹 생성, 선택, 이름 변경, 삭제
- 그룹 간 작업 이동
- 새 작업 추가 및 수정
- 우선순위, 기한, 반복 설정
- 작업 완료 및 복원
- 완료 목록 접기 / 펼치기
- 완료 항목 삭제 및 보관
- 보관함에서 기록 확인, 복원, 삭제
- 작업 복제 및 순서 정리
- 우클릭 메뉴를 이용한 빠른 작업 처리
- 다중 선택 및 일괄 정리
- 트레이 연동
- 바탕화면 고정
- 설정을 통한 동작 방식 조정
- 자동 저장 및 백업 / 복원

---

## 사용 환경

ToDoDo는 다음과 같은 사용자에게 잘 맞습니다.

- 프로젝트별로 작업을 나누어 관리하고 싶은 경우
- 완료한 작업도 기록으로 유지하고 싶은 경우
- 반복되거나 유사한 작업이 많은 경우
- Windows 데스크톱에서 항상 가까운 위치에 두고 작업을 관리하고 싶은 경우
- 단순 메모형 앱보다 구조적인 작업 관리 도구를 원하는 경우
